## WebMCP Research

**Announcement:** Feb 10, 2026 - Google Chrome blog
**Status:** Early preview program only
**What it does:** Standard way for websites to expose structured tools to AI agents

**Two APIs:**
1. Declarative - standard actions defined in HTML forms
2. Imperative - complex dynamic interactions requiring JavaScript

**Relevance:** Could solve headless browsing fragility if websites adopt it
**Catch:** Chicken-and-egg problem - websites won't add it until agents use it
